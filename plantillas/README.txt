PLANTILLAS EXCEL PARA EL SISTEMA DE NÓMINA VILLA VENECIA

Este directorio contiene las plantillas Excel que el sistema utiliza para generar reportes.

ARCHIVOS REQUERIDOS:

1. CUADRE_CAJA.xlsx
   - Plantilla para el cuadre de caja diario
   - Debe contener encabezados en las celdas especificadas
   - El sistema la llenará automáticamente con los datos

2. NOMINA.xlsx
   - Plantilla para la nómina quincenal
   - Debe contener empleados predefinidos en las filas B12-B22
   - El sistema calculará y llenará los valores automáticamente

NOTA: Si no tienes estas plantillas, el sistema las creará automáticamente
con la estructura básica necesaria.

Para crear tus propias plantillas personalizadas:
1. Copia estos archivos
2. Modifica el formato y diseño según tus necesidades
3. Mantén los encabezados en las posiciones especificadas
4. El sistema seguirá funcionando con tus plantillas personalizadas
